Basic Widget Styling

Header

'box-padding' padding value is '30px'

'box-padding narrow' padding value is '15px'

'box-padding narrow-horizontal' padding value is '15px 30px'

'box-padding narrow-vertical' padding value is '30px 15px'

Need instructions?

  1. Widget - Define div with 'widget' class. 'widget' have 30px margin-bottom.
  2. Background Color - Define a div to have a background color. Starts with 'bg-color' and color that you choose. If the box is at top, you can add 'rounded-top' and 'rounded-bottom' if down below. If it is only one box, then you can put 'rounded'.
  3. Padding - Define 'box-padding' if you want to include padding. Padding variations are as listed above.
