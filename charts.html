Inline Sparkline

Inline sparklines take their values from the contents of the html page

HTML Codes
						<span class="inlinesparkline">5,6,7,2,0,-4,-2,-6</span>
JS Codes
						$('.inlinesparkline').sparkline('html', {
				        	type: 'bar',
				        	height: '100',
				        	barWidth: 10, 
				        	barColor: '#4fa950',
						    negBarColor: '#ce483d',
						    stackedBarColor: '#FFA93C',
				        } );
5,6,7,2,0,-4,-2,-6

Dynamic Sparkline

Sparklines can also take their values from the first argument passed to the sparkline() function

HTML Codes
						<span class="dynamicsparkline">loading...</span>
JS Codes
						var myvalues = [10,8,5,7,4,4,1];
				        $('.dynamicsparkline').sparkline(myvalues, {
				        	type: 'bar',
				        	height: '100',
				        	barWidth: 10, 
				        	barColor: '#4fa950',
						    negBarColor: '#ce483d',
						    stackedBarColor: '#FFA93C',
				        });
